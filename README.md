## 프로젝트 개요

* 사이트 주소

* **Movingo**는 유저 기반의 영화 추천 및 리뷰 커뮤니티이다. 빙고게임을 활용해 유저의 적극적인 리뷰 기록을 유도하고, 주제별 영화 컬렉션과 영화 별 커뮤니티 생성을 통해 유저들의 소통을 돕는다.

* 주제 : 영화 추천 커뮤니티

* 개발 기간 : 2022년 5월 20일 ~ 2022년 5월 27일
